# 🚀 SkillSage

SkillSage is an AI-powered full-stack learning and interview preparation platform built using the MERN stack with GenAI integration.

It helps users upload resumes, analyze job descriptions, identify skill gaps, and generate AI-powered interview questions and ATS-optimized resumes.

---

## ✨ Features

- JWT Authentication (Login/Register)
- User Management System
- Resume Upload & Processing
- AI-based Skill Gap Analysis (Gemini AI)
- AI Interview Question Generation
- ATS Optimized Resume Generation
- PDF Generation using Puppeteer
- Secure Token Handling

---

## 🧠 AI Integration

SkillSage uses Google Gemini AI to:
- Analyze resumes
- Extract skills
- Compare job descriptions
- Generate interview questions
- Improve ATS resume quality
